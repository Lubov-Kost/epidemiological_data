# Проект epidemiological_data
Файл *`main.py`* - бекенд на python, принимет в режиме около реального времени с задаваемой частотой дискретизации

Папка *`datasets`* содержит два датасета, связанных с эпидемиологией (частоты заболеваемости за период и гео-метки) 

Файл *`plotly_test.py`* - тестовый график с географическими координатами
